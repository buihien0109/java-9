## Ứng dụng quản lý user

### Đối tượng (User)

Các thuộc tính của đối tượng user bao gồm:
- id
- name
- email
- phone
- address
- avatar
- password

### Chức năng

- **Trang index**
    - Hiển thị danh sách user
    - Xóa user
    - Tìm kiếm user theo tên

- **Trang create**
    - Thêm user

- **Trang detail**
    - Xem chi tiết user
    - Đổi avatar
        - Upload image
        - Lấy danh sách image
        - Tìm kiếm image
        - Xóa image
        - Xem image
        - Download image
        - Phân trang
    - Đổi password
    - Quên password
    - Cập nhật thông tin user
